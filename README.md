# Timbiriche
Multiplayer game of Timbiriche using Node.js and socket.io

## Usage
To install the necesary modules:
```bash
$ npm install
```
To start the server
```bash
$ node server.js
```
The using your browser connect to
```bash
127.0.0.1:8080
```
